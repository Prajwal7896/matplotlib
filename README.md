# Farmers Market Data Analysis

This project analyzes farmers market data using Python, Pandas, and Matplotlib.

## Features
- Crop price trend analysis
- Demand analysis using bar graphs
- Profit comparison and monthly profit trends

## Technologies Used
- Python
- Pandas
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run files from src folder:
   python crop_price_trends.py
   python demand_analysis.py
   python profit_analysis.py
